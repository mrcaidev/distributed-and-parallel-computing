# 分布式并行计算

电子科技大学 2020 级《分布式并行计算》课程代码。

## ✨ 项目介绍

1. 埃拉托斯特尼筛法性能优化（MPI）
    - 去除偶数
    - 消除广播
    - 提高 cache 命中率
2. N-body 问题性能优化（Cuda）
    - 并行化
    - 手动管理内存
    - 线程共享内存
